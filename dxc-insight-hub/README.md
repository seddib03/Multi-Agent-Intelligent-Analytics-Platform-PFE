# DXC Insight Hub

Application front-end construite avec Vite, React, TypeScript, shadcn/ui et Tailwind CSS.

## Prerequis

- Node.js 18+
- npm 9+

## Lancer en local

```sh
npm install
npm run dev
```

Le serveur de developpement demarre sur `http://localhost:8080`.

## Scripts utiles

- `npm run dev`: lance le serveur de developpement
- `npm run build`: genere le build de production
- `npm run preview`: lance un serveur de previsualisation du build
- `npm run lint`: execute ESLint
- `npm run test`: lance les tests Vitest
- `npm run test:watch`: lance les tests Vitest en mode watch

## Stack technique

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
