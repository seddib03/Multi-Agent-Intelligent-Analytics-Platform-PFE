# Frontend App

## Overview

This frontend is built with React + TypeScript and uses Vite for development and production builds.

## Prerequisites

- Node.js 18+
- npm 9+

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Preview Production Build

```sh
npm run preview
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
